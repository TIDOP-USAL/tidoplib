Descripción detallada

template<typename T>
class I3D::Segment< T >

Clase segmento 2D.

Esta template representa un segmento recto entre dos puntos independientemente de que sean int, double o float. Se utiliza la clase point_ de OpenCV.

Se han definido los siguientes alias para facilitar el acceso:

typedef Segment<int> SegmentI;
typedef Segment<double> SegmentD;
typedef Segment<float> SegmentF;
typedef SegmentI Line;
