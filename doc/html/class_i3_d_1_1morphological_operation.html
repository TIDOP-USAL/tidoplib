Descripción detallada

Clase base para las operaciones morfológicas.

Operaciones morfologicas básicas:

  • Erosion (Erotion)
  • Dilatacion (Dilate)

Operaciones morfologicas avanzadas (cv::morphologyEx):

  • Apertura (Opening) = erosion + dilation
  • Cierre (Closing) = dilatación + erosión
  • Morphological Gradient = dilatación - erosión
  • TopHat = src - apertura(src)
  • BlackHat = cierre(src) - src
