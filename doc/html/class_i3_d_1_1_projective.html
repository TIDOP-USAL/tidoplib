Descripción detallada

template<typename T>
class I3D::Projective< T >

Transformación Projectiva.

La Transformación Projectiva expresa la relación que existe entre dos planos.

x' = ( a * x + b * y + c ) / ( g * x + h * y + 1 )
y' = ( d * x + e * y + f ) / ( g * x + h * y + 1 )
