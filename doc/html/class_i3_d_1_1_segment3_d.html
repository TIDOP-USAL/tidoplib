Descripción detallada

template<typename T>
class I3D::Segment3D< T >

Clase segmento 3D.

Esta template representa un segmento recto entre dos puntos independientemente de que sean int, double o float. Se utiliza la clase Point3_ de OpenCV.

Se han definido los siguientes alias para facilitar el acceso:

typedef Segment3D<int> Segment3dI; typedef Segment3D<double> Segment3dD; typedef Segment3D<float> Segment3dF;
