40// Generalized Hough transform:
41//
42// The Generalized Hough Transform or GHT, introduced by Dana H. Ballard in 1981,
43// is the modification of the Hough Transform using the principle of template matching.
44// This modification enables the Hough Transform to be used for not only the detection
45// of an object described with an analytic function. Instead, it can also be used to
46// detect an arbitrary object described with its model.
47// https://en.wikipedia.org/wiki/Generalised_Hough_transform
48// http://www.cs.utexas.edu/~dana/HoughT.pdf
