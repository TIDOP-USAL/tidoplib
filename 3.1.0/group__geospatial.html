◆ inverseFlattening()

auto tl::Ellipsoid::inverseFlattening ( ) const -> double
inline

Inverse flattening.

\[ 1/f = \frac{a}{a - b} \]

https://docs.ogc.org/is/18-010r7/18-010r7.html#66 ISO 19111 allows an oblate ellipsoid to be defined through semi-major axis (a) and either semi-minor axis (b) or inverse flattening (1/f). If semi-minor axis is used as the second defining parameter the value for inverse flattening to be shown in the WKT string should be calculated from 1/f = a / (a – b).

ISO 19111 also allows for the earth model to be a sphere, for which 1/f is infinite. In this document if the earth model is a sphere <inverse flattening> shall be given an artificial value of zero.
