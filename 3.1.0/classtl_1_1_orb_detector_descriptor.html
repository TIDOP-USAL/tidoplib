auto scaleFactor () const -> double override
 Pyramid decimation ratio ScaleFactor==2 means the classical pyramid, where each next level has 4x less pixels than the previous, but such a big scale factor will degrade feature matching scores dramatically. On the other hand, too close to 1 scale factor will mean that to cover certain scale range you will need more pyramid levels and so the speed will suffer.
 
auto levelsNumber () const -> int override
 Returns the number of pyramid levels The smallest level will have linear size equal to input_image_linear_size/pow(scaleFactor, nlevels - firstLevel)
