auto threshold () const -> int override
 threshold The AST applies a minimum difference threshold when comparing the value of a pixel on the pattern with the brightness of the nucleus. This parameter controls the sensitivity of the corner response. A large threshold value results in few but therefore only strong corners, while a small threshold value yields also corners with smoother gradients.
