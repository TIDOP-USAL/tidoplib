Detailed Description

Class to measure times.

If a message is set, when the stop method is called, a message is printed to the console. Otherwise, the stop method only returns the time in seconds.

Example:

Time measurement with message:

Chrono chrono("Translation");
chrono.run();
chrono.stop(); // Print time in console with a message and returns time in seconds
