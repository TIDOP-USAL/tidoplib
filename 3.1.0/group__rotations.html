Converts a rotation matrix to quaternions.

Implementación RotationConverter.

\[ x^2 = \frac{+r_{00} - r_{11} - r_{22} + 1}{4} \]

\[ y^2 = \frac{-r_{00} + r_{11} - r_{22} + 1}{4} \]

\[ z^2 = \frac{-r_{00} - r_{11} + r_{22} + 1}{4} \]

\[ w^2 = \frac{+r_{00} + r_{11} + r_{22} + 1}{4} \]

\[ x^2 + y^2 = \frac{1 - r_{22}}{2} \]

\[ z^2 + w^2 = \frac{1 + r_{22}}{2} \]

\[ y^2 - x^2 = \frac{r_{11} - r_{00}}{2} \]

\[ w^2 - z^2 = \frac{r_{11} + r_{00}}{2} \]

\[ x*y = \frac{r_{01} + r_{10}}{4} \]

\[ x*z = \frac{r_{02} + r_{20}}{4} \]

\[ y*z = \frac{r_{12} + r_{21}}{4} \]

\[ x*w = \frac{r_{21} - r_{12}}{4} \]

\[ y*w = \frac{r_{02} - r_{20}}{4} \]

\[ z*w = \frac{r_{10} - r_{01}}{4} \]
